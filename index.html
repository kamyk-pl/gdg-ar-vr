Glossary

VR - Virtual Reality

  • stereoscopic view

  • headset is required

  • view is rendered by device

  • 3-DoF or 6-DoF

AR - Augmented Reality

  • real world
  • extended by computer generated content
  • headset or mobile device required

MR - Mixed/Merged Reality

  • Microsoft Hololens
  • The Void

WebXR

  • WebVR (2014-2018)
  • Immersive Web  working group at W3C
  • Misson: "We help bring high-performance Virtual Reality (VR) and Augmented Reality (AR) (collectively known as XR) to the open Web via APIs to interact with XR devices and sensors in browsers."
  • WebXR Device API - low level JavaScript API which exposes the VR platform capabilities

web vs native apps

pros

  • no installation
  • no mobile stores restrictions
  • fast implementation (one app all platforms)
  • low entry threshold
  • instant updates

web vs native apps

pros

  • security
  • 3d content can be indexed and searched
  • extend current websites
  • it is our playground with our tools
    (Dev Tools, Npm, VisualCode, ...)

web vs native apps

cons

  • it is not yet production ready
  • offline resources
  • performance
  • web limitation (saving files, offline mode, ...)

Challenges

  • lack of 3D models

    Poly,  SketchFab

  • costs of 3D models

    > 30 000 $

Challenges

  • size of models

    > 2,5MB

  • lack of UI guidelines and good practices
