Glossary

VR - Virtual Reality

  • stereoscopic view

  • headset is required

  • view is rendered by device

  • 3-DoF or 6-DoF

AR - Augmented Reality

  • real world
  • extend by computer generated content
  • headset or mobile device required

MR - Mixed/Merged Reality

  • Microsoft Hololens
  • The Void

Web XR

  • The first proposed standard of  Immersive Web  working group at W3C
  • low level JavaScript API which exposes the VR platform capabilities
  • successor of WebVR
  • Goal: unify 3D immersive experiences using the Web

Web XRW3C Status

  • W3C receives a Submission
  • W3C publishes a Note
  • W3C creates a Working Group
  • W3C publishes a Working Draft
  • W3C publishes a Candidate Recommendation (expect 12.2019)
